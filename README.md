# Simple C Programming Statistics Project

Author: Vladislav Skachkov
Date: 05.11.2025

## Description of the Project

This repository contains a simple C program /stats.c/ where the statistical analysis on a dataset of unsigned char values is performed. It computes the minimum, maximum, mean, median, and sorts the data from largest to smallest.

## Files Included 

- **stats.c** - Implementation file with function definitions.
- **stats.h** - Header file with function declarations.
- **README.md** - Description of the project and author information.

## Compilation Stage

~bash

Run these commands in your Terminal:

	gcc -o stats.out stats.c 

	./stats.out
